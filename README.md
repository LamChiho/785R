## Getting Started

### Prerequisites

- Python 3.8 or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/LamChiho/785R.git
   cd 785R
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Project

```bash
python src/main.py
```

### Running Tests

```bash
python -m pytest tests/
```
